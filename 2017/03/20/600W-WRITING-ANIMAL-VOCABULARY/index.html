[600W][WRITING] Animal Vocabulary

2017-03-20
English
600W, English, writing

Baby

  1. kitten - 새끼 고양이
  2. puppy - 새끼 개
  3. kid - 새끼 염소(goat)
  4. foal - 망아지, 새끼 당나귀
  5. calf - 송아지
  6. cub - 곰, 사자, 여우 새끼
  7. chick - 병아리

group

  1. pride - 사자 무리
  2. flock - 양, 염소, 새 떼
  3. murder - 까마귀 떼
  4. school - 물고기, 해양동물의 떼
  5. pack - (함께 사냥을 하거나 하기 위해 모인 동물들) 무리
  6. herd - 소, 사슴, 양, (함께 살고 함께 먹이를 먹는 동종 짐승의) 떼
  7. swarm - 벌, 곤충의 떼

class

  1. reptile - 파충류
  2. mammal - 포유류
  3. fish - 어류
  4. crustacean - 갑각류
  5. insect - 곤충류
  6. bird - 새
  7. amphibian - 양서류

movement

  1. gallop - (말 등이) 전속력으로 달리다.
  2. fly - 날다
  3. swim - 수영하다
  4. crawl - 기어가다.
  5. slither - (뱀이) 매끄럽게 스르르 나가아다.
  6. hop - 깡총 깡총 뛰다.
  7. run - 뛰다.

sound

  1. hiss - (뱀이) 쉬익 소리를 내다.
  2. bark - (개, 여우가) 짖다.
  3. purr - (고양이가) 가르랑 거리다.
  4. buzz - (벌이) 윙윙 거리다.
  5. growl - (개가) 으르렁 거리다.
  6. roar - (큰 짐승이) 포효하다.
  7. hoot - (부엉이가) 부엉 부엉.
  8. sing - (새가) 지저귀다.
  9. howl - (개나 늑대가) 길게 울다.

cover

  1. shell - 단단한 껍질
  2. feather - (새의) 깃털
  3. fur - 털
  4. scales - 비늘
  5. skin - 피부, 동물가죽, 뱀의 허물.

something on the head

  1. antler - (사슴의) 가지있는 뿔
  2. antenna - 더듬이
  3. tusk - 코끼리 엄니
  4. whiskers - (고양이, 쥐의) 수염
  5. snout - (돼지,개의) 주둥이
  6. horn - (소,양들의) 뿔
  7. beak - (새의) 부리

limb

  1. talon - (맹금류의) 발톱
  2. paw - (발톱이 달린 동물의) 발
  3. hoof - 발굽
  4. fin - 지느러미
  5. wing - (새,곤충의) 날개
  6. claw - (동물, 새의) 발톱, (게의) 집게발

actions

  1. hunt - 사냥하다
  2. migrate - (새/동물)이 계절에 따라 이동하다.
  3. hibernate - 동면하다.
  4. lay - (새/곤충/어류)가 알을 낳다.
  5. procreate - 새끼를 낳다.
  6. graze - 풀을 뜯어 먹다.
  7. wag - 개가 꼬리를 흔들다.

habitat

  1. wild - 야생
  2. domestic - 사육의
    • domesticate - 사육하다.
  3. tame - 야생에서 잡혀 길들여진.
  4. feral - 사육되다가 야생으로 돌아간.

what they eat

  1. herbivore - 초식 동물
  2. omnivore - 잡식 동물
  3. carnivore - 육식 동물
